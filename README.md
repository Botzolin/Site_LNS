## git add . (adauga fisierele)
## git commit -m "Am modificat headerul" (commit)
## git push -u origin nume-feature (push)


Detalii Importante:

##  Pentru git pull trebuie accesat initial branchul main -> git checkout main

# Workflow:
## git checkout DevSpace (aduce DevSpace pe Pc-ul tau)
## git pull (aduce modificarile)
## git checkout feature/numele branchului tau (intri pe branchul personal)
## git merge DevSpace (copiezi in branch ul tau ultima versiune din dev ca sa fim toti la zi)

## git add . (adaugi ce ai lucrat)
## git commit -m "update" (dai comit cu comentariu)
## git push (se da push automat pe branchul tau daca lucrezi deja in el)

## Dupa mergi pe github la pull requests si creezi un pull request nou. Trebuie sa selectezi in chenare din ce branch in ce branch vei da requestul (trebuie din branchul tau in DevSpace). Dupa aceea poti da merge daca totu e ok. La final se da pull request din DevSpace in main.
